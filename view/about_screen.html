File Storage site


BUT FIT, Web application project 2



This site was created as a project for WAP. Its functionality is to upload, store and manage files on a server. For our backend we use Express api. The file storage is managed by MongoDB using Mongoose library. Files are stored localy on the server with metadata stored in mongodb for easy management. Files are also periodicaly cheked and old files are deleted from the server.

You can upload you file by clicking on upload section in navigation bar or just bu clicking on the logo. After upload, server will generate two unique links. Its up to you as a user to store them or distribute them as you need.
Firt (download) link will enable you to download the uploaded file.
Second (manage) link will open up a manage screen where you can: delete file, generate new link, check download count or redownload the file.

Technologies used for this project: Express, Mongooose, Multer, Bootstrap.
